TLACKWALI

Es una aplicación web desarrollada para el sector de restaurantes.

Sobre el proyecto

Esta aplicación web cuenta con dos modos, el de administración de productos y servicios, y el de tipo tienda-menú en línea, en donde los clientes podrán tener una cuenta y levantar pedidos de lo que desean consumir.

Inicio de sesión


Esta aplicación web cuenta con autenticación de usuarios en donde los empleados del sitio y los clientes pueden iniciar sesión para poder administrar el negocio.

Página de inicio


En esta página de inicio el cliente tiene una vista general de los productos divididos por categorías que el establecimiento está ofreciendo.

Lista de productos


Como podemos observar, el producto promocionado en esta parte destaca con una imagen grande y una descripción del mismo para llamar la atención del comensal.

Lista de productos


Esta sección tiene también distintos apartados en donde el restaurante puede hacer micro promociones de sus platillos más demandados o bien, aquellos que son recién integrados al menú.

Perfil de usuario


Esta sección reúne toda la información acerca del comensal que se ha registrado en la plataforma, de esta manera puede ver y gestionar sus pedidos, así como ver el consumo promedio en el establecimiento.

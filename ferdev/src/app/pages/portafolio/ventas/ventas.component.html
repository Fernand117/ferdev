SISTEMA PUNTO DE VENTAS DE AUTO PARTES

SPVAP

Es una aplicación web especialmente para el sector de las autopartes.

Sobre el proyecto

Esta aplicación web permite gestionar desde un negocio pequeño a través de diferentes módulos con los cuales el usuario podrá tener el control total sobre las ventas de su negocio.

Inicio de sesión


Esta aplicación web cuenta con autenticación de usuarios en donde los empleados del sitio pueden iniciar sesión para poder administrar los productos, pedidos y almacén del negocio.

Página de inicio


En la página de inicio podremos tener una lista general de los productos añadidos recientemente así como también tres paneles que muestran información general sobre la cantidad total de productos, remisiones y clientes.

Lista de productos


Esta vista está diseñada al estilo de tienda en línea, donde podremos observar una foto del producto así como detalles del mismo tales como su código, descripción y precio.

Lista de productos


Esta sección cuenta con una tabla con datos similares a la vista anterior, sin embargo aquí contaremos con funciones de administrador, las cuales permitirán editar o eliminar algún producto.

Administración de productos


Como podemos observar, la aplicación web cuenta también con una vista que permite añadir un nuevo producto o bien editarlo.

Lista de compras


La aplicación web permite también llevar un control de las ventas realizadas a los proveedores del negocio, desde aquí podremos gestionar el ingreso al almacén de algún producto, en donde indicaremos la cantidad y la fecha de ingreso.

Lista de remisiones


La aplicación web permite emitir remisiones suplantando por completo la remisión a papel, de esta forma tendremos de manera segura cada una de ellas en nuestra base de datos, cabe señalar que estas remisiones son generadas en automático con tan solo un click.

Detalles de remisión


Aquí podremos ingresar todos los detalles relacionados con la remisión que queremos emitir, tales como el cliente al que va dirigida, los productos que se desean adquirir y la propia emisión del ticket.

Detalles de remisión


Como podemos observar, en los detalles de la remisión podemos añadir el o los productos que nuestro cliente desea comprar, aquí se nos mostrarán los detalles tales como el precio antes de IVA, precio con IVA, el Subtotal y el Total.

Ticket


Por último contamos con la generación del ticket, este tiene soporte para poder ser impreso en mini impresoras para tickets y en formato PDF.

SIVI WALD

SIVI es una aplicación web desarrollado para la empresa industrias wald.

Sobre el proyecto

Esta aplicación web está dirigida hacía los vendedores de la empresa, esto con la finalidad de poder llevar un control de los víaticos que les son asginados.

Inicio de sesión


Esta aplicación web cuenta con autenticación de usuarios en donde solamente vendedores y el encargo del área puede iniciar sesión para poder ver los registros generados por los propios vendedores.

Página principal


En la vista principal poremos encontrar información general acerca de los vendedores y del registro de cada factura que van ingresando al sistema. La aplicación web permite subir archivos tanto del tipo imagen como PDF para posteriormente ser descargados.

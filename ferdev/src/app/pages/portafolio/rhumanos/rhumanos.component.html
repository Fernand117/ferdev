SISTEMA DE RH

SRH

El sistema de RH es un software de escritorio desarrollado para dicho departamento de la empresa Industrias Wald.

Sobre el proyecto

La finalidad de este proyecto es poder llevar un registro de los empleados que laboran en dicha empresa, dentro del sistema se almacena información personal de cada empleado, así como también salidas, el historial, etcétera.

Inicio de sesión


Este sistema así como muchas de las aplicaciones móviles y web que desarrollamos cuenta con un sistema de autenticación de usuarios, esto para prevenir el uso mal intencionado del mimso por parte de personal no autorizado.

Vista principal


En la vista principal nos encontramos con un resumen de lo que se puede realizar dentro del sistema. A un costado nos encontramos con un menú de opciones

Administradores


En este apartado podemos registrar a un nuevo adminsitrador, del cual, el sistema nos pide sus datos básicos, este administrador pasa también a formar parte de la lista de empleados.

Personal


Aquí podemos registrar a los empleados, dentro de la información que se registra está el Número de Seguridad Social, el área a la que pertenece, una foto de el/ella, entre otros datos más de interés para la empresa.

Permisos


El sistema permite llevar un registro de los permisos que se le van otorgando a cada uno de los empleados, la fecha en que se emitió y la fecha en que vence el permiso.

Historial


El sistema también permite llevar un historial sobre el comportamiento de cada uno de los empleados, esto para poder definir accines futuras a tomar si el comportamiento del mismo es negativo para la empresa.
